# 社区行为准则

为营造开放、尊重、安全的技术交流环境，本仓库参与者须遵守以下约定：

## 我们的承诺

无论年龄、体貌、民族、经验水平、教育程度、社会身份与技术栈如何，我们致力于为每个人提供 **无骚扰、无歧视** 的参与体验。

## 不可接受的行为

包括但不限于：

- 使用性暗示、暴力或歧视性语言与图像；  
- 骚扰、人肉、恶意跟踪或威胁他人；  
- 在公开渠道发布他人隐私或未授权的商业机密；  
- 将本仓库用于或鼓吹 **违法破解、未授权入侵、盗版传播**；  
- 其他在专业场合中明显不当的行为。

## 执行

维护者可对违反本准则的 Issue、评论、PR 或账号采取 **删除内容、拒绝合并、临时或永久禁止参与** 等措施。情节严重或涉及违法的可向平台方举报。

本准则以常见开源社区实践为蓝本编写；若你希望采用更正式的 Contributor Covenant 全文，可自行替换并注明版本号。
